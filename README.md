# ALMADERY

Fine Diamonds & Gold Jewellery — cinematic scroll-driven teaser landing page.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- GSAP + ScrollTrigger

## Scripts

```bash
npm install
npm run dev      # local development
npm run build    # production build
npm start        # serve dist on port 3000
npm run preview  # vite preview
```

## Experience

A single pinned scroll journey:

**Rough Diamond → Polished Diamond → Gold Craftsmanship → Diamond Setting → Finished Engagement Ring → ALMADERY Brand Reveal → Private Access**

Motion is lightened on mobile and replaced with restrained fades when `prefers-reduced-motion` is enabled.
