# ALMADERY

Fine Diamonds & Gold Jewellery — cinematic scroll-driven teaser landing page.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- GSAP + ScrollTrigger

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Experience

A single pinned scroll journey:

Rough Diamond → Polished Diamond → Gold Craftsmanship → Diamond Setting → Finished Engagement Ring → ALMADERY Brand Reveal → Private Access
