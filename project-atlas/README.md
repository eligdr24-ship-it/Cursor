# Project Atlas — Preview

Institutional equity research workstation for finding long-duration compounders before they become consensus.

This folder contains a **UI preview only**. It does not generate live research, pull market data, or give investment advice.

## Run the preview

```bash
cd project-atlas
npx --yes serve public -p 4173
```

Then open `http://localhost:4173`.

## What’s included

- Brand landing composition
- Decision-gate checklist
- Sample company dossier (illustrative data)
- Conviction score, bull/base/bear cases, metrics monitor

## What’s not included yet

- Live financial data / filings ingestion
- AI research generation
- Screening universe + ranking
- Portfolio of ≤20 positions
- Thesis monitoring / alerts
- Auth, persistence, multi-user workspace
