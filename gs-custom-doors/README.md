## Print notes

1. Print at **100%** — do not scale.
2. Prefer **TRIM** files for home/office printers.
3. Prefer **bleed** files for professional print shops.
4. Artwork is RGB @ 300 DPI; ask the shop to convert to CMYK if needed.
5. See `print-ready/PRINT-SPECS.txt` for full specs.
